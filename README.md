# Patient Dataset Project using NumPy and Pandas

## Description
This project demonstrates the creation and manipulation of a patient dataset using **Python**, **NumPy**, and **Pandas**.  
It focuses on generating structured patient data, performing basic data analysis, and showcasing how Python libraries can be used for healthcare data management.

## Features
- Generate a synthetic patient dataset with fields like:
  - Patient ID
  - Name
  - Age
  - Gender
  - Medical Condition
  - Admission Date
- Perform basic data operations using **Pandas**:
  - Filtering and sorting data
  - Aggregation and summary statistics
  - Data visualization (optional)
- Easy to modify and extend for additional fields or analysis
